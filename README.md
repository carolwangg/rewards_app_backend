# <img width="30" height="30" alt="zinks" src="https://github.com/user-attachments/assets/b240ff94-c8f9-41c4-b319-f11803e5d870"/> Zinks Backend Web Server
Code for running the web server on which the Zinks app operates.

## Contributor Setup
### Dependencies
- Node.js
- npm
- git
- MySQL Server
- MySQL CLI or MySQL Workbench

### Steps
1. Clone the repo
```
git clone https://github.com/carolwangg/rewards_app_backend.git
```
3. Open the repo in VSCode
4. Install dependencies by opening terminal and running
```
npm install
```
